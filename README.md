Welcome to: The VAIIYA website project! 
The VAIIYA website is a website developed, hosted, and paid for by ME!(T342 >;3)

If you want to support me(T342) get in touch via discord, and send me a DM request through servers I am in!

## SITE INFO:

MAIN hosting location: https://www.vaiiya.ca/


PREVIEW DEPLOY LOCATION: https://pre.vaiiya.ca/


BRANCH ORDER OF DEPLOYED STAGES:\
MASTER - The head deploy for the head domain, https://www.vaiiya.ca/ \
PREVIEW - The 2nd deploy branch, this is where all new changes go before they are pushed to the main branch. these changes get deployed to https://pre.vaiiya.ca/
